# `<liist-daad-filter>`

A component that displays an avatar image and the name of the author.


## Properties

- **`serviceTypes`**: Array of strings. Each item represents a `<liist-toggle-bttn>`. Can be dynamically set with `setServiceTypes(array)`.
- **`categories`**: Array of strings. Each item represent a `<liist-check-item>`. Can be dynamically set with `setCategories(array)`.
- **`device`**: either "desktop" (default) or "mobile". Events triggered depend on device type.


## API (Custom Events)

**DESKTOP**
On every toggle the component sends a custom `liist-daad-filter-changed` event. Which can be catched with the following code (however this does not work for storybook showcase, but it works in a real usecase):
```javascript
{
  selected: {
    categories: ["american", "sushi"], // array with categories selected
    serviceTypes: ["delivery"],        // array with delivery options
  }
}
```

**MOBILE**
On every press of the action buttons (apply / cancel) the component triggers a `liist-daad-filter-action` event, which holds the following information:
```javascript
{
  action: "apply", // or "cancel"
  selected: {
    categories: ["american", "sushi"], // array with categories selected
    serviceTypes: ["delivery"],        // array with delivery options
  }
}
```

**SEE IN ACTION**
This snipped listens to all events triggered by the component and outputs the passed data in the console.
```javascript
document.querySelector("liist-daad-filter").addEventListener("liist-daad-filter-changed", (e) => {
  console.log(e.detail);
});
document.querySelector("liist-daad-filter").addEventListener("liist-daad-filter-action", (e) => {
  console.log(e.detail);
});
```
