# `<liist-section>`

liist section component used to structure the web view

### Customized State
<Preview>

    <span slot="fill-content">
      <div id="sample-content">
        <li>This is a span with a slot <i>fill-content</i></li>
        <li>this slot can be filled with any html and styled respectively</li>
      </div>
    </span>
  </liist-section>

## Properties
There are no properties for this component but 2 Slots of which the content can be changed
- **`slot="title"`**: title of the section. p tag
- **`slot="fill-content"`**: can be filled with any content
