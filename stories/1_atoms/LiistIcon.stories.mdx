# `<liist-icon>`

All icons.

## Properties

| Property | Description | Default |
| -------- | ----------- | ------- |
| `icon`   | name of the icon, se a dictionary of all names below! | `pin` |
| `color`  | Color of the stroke. LiistColors or any valid color input is supported. | `smokii` |
| `size`   | Icon size, specified in px or percentage (e.g. `size="50px"` or `size="100%"`) | `28px` |
