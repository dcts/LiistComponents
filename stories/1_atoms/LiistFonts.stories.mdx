# LiistFonts

Liist uses 2 fonts: [**Sporting Grotesque**](https://velvetyne.fr/fonts/sporting-grotesque/) and [**IBM Plex Sans**](https://fonts.google.com/specimen/IBM+Plex+Sans?selection.family=IBM+Plex+Sans:wght@700#standard-styles). They are loaded and stored as **`global CSS variables`**:

| Font                   | CSS Variable                | `font-weight` |
| ---------------------- | --------------------------- | ------------- |
| **Sporting Grotesque** | `var(--liist-display-font)` | 400 regular |
| **IBM Plex Sams**      | `var(--liist-main-font)`    | 400 regular<br></br>500 medium<br></br>600 semi-bold<br></br>700 bold |

## Usage (with components library)
The fonts need to be loaded once, this is best achieved with an injected script tag inside the HEAD of the HTML:
```html
<html>
<head>
  <meta charset="utf-8">
  <script type="module">
    import 'liist-webcomponents/src/atoms/LiistFonts.js';
    import { LiistBttn } from 'liist-webcomponents/src/atoms/LiistBttn.js';
  </script>
</head>
<body>
  <liist-bttn text="works?"></liist-bttn>
</body>
</html>
```

## Usage (with HTML)
You can import the fonts and the global CSS variables in any webproject by adding the following two `link` tags in the HEAD of your HTML:
```html
<html>
<head>
  <meta charset="utf-8">
  <link href="https://liistwebfonts.imfast.io/css/import-fonts.css" rel="stylesheet" type="text/css"
    crossorigin="anonymous">
  <link href="https://liistwebfonts.imfast.io/css/define-fonts.css" rel="stylesheet" type="text/css"
    crossorigin="anonymous">
</head>
<body>
  <style>
    /* styling for demopage */
    .test { font-size: 40px; }
  </style>
  <p style="font-family: var(--liist-display-font);" class="test">Sporting Grotesque</p>
  <p style="font-family: var(--liist-main-font); font-weight: bold;" class="test">IBM Plex Sans (bold)</p>
  <p style="font-family: var(--liist-main-font);" class="test">IBM Plex Sans</p>
</body>
</html>
```
