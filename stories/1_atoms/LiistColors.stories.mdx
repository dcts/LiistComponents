# `LiistColors`

Liist brand colors. Colors are supported widely across all components. Whenever a component has a property `color` or `bgColor` then you can simply add the lowercase name of the liist brand-color. For example:
```html
<liist-component color="sunii" bgColor="hotii"><liist-component>
```

### Initialization

```js
Import { LiistColors} from 'liist-components/src/atoms/LiistColors.js';
LiistColors.initCssVariables();
```

### Themes

A theme is always a combination of 2 brand colors and always consists 2 properties:
- **`bgColor`**: main color of the theme
- **`color`**: font color

Themes are supported among these components:
- `<liist-bttn>`

```js
  viiolet80:  { bgColor: "viiolet80", color: "viiolet20" },
  viiolet80b: { bgColor: "viiolet80", color: "creamii" },
  sunii:      { bgColor: "sunii",     color: "viiolet80" },
  piink:      { bgColor: "piink",     color: "hotii" },
  grasii:     { bgColor: "grasii",    color: "piink" },
  hotii:      { bgColor: "hotii",     color: "creamii" },
  viiolet:    { bgColor: "viiolet",   color: "sunii" },
}
```

### API (functions)

| Function                         | Description |
| -------------------------------- | ----------- |
| `LiistColors.names()`            | returns all color names as array. |
| `LiistColors.get(color)`         | input color name, return color hex as string. |
| `LiistColors.values()`           | returns all color hex values as array. |
| `LiistColors.themes()`           | returns all theme names. |
| `LiistColors.getTheme(theme)`    | input theme name, return theme object storing `color` and `bgColor` of this theme as hex. |
| `LiistColors.dict`               | returns dictionary object storing all colors. |
| `LiistColors.themeDict`          | returns dictionary object storing all themes. |
| `LiistColors.isValidTheme(name)` | returns boolean. Checks if themename is valid. |
