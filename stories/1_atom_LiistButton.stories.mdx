# `<liist-bttn>`

Main button component.

## Properties

- **`text`**: text content of the button
- **`color`**: text color of button. **IMPORTANT**: Only applied when `theme="custom"`.
- **`bgColor`**: background color of the button. **IMPORTANT**: Only applied when `theme="custom"`.
- **`theme`**: theme of the button, see [LiistColors documentation](?path=/docs/atoms-liistcolors--page). **Default**: `viiolet80`. **IMPORTANT**: overrides `color` and `bgColor`.
- **`width`**: width set in percentage or pixels. For example `250px` or `100%`. **Default**: `100%`
- **`url`**: url as string. If no url is set, button is not clickable!
- **`openInNewTab`**: opens url in new tab. only applied when url is set.
