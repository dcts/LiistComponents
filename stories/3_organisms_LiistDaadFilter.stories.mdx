# `<liist-daad-filter>`

A component that displays an avatar image and the name of the author.


## Properties

- **`serviceTypes`**: Array of strings. Each item represents a `<liist-toggle-bttn>`. Can be dynamically set with `setServiceTypes(array)`.
- **`categories`**: Array of strings. Each item represent a `<liist-check-item>`. Can be dynamically set with `setCategories(array)`.


## API (Custom Events)

On every toggle the component sends a custom `liist-daad-filter-changed` event. Which can be catched with the following code (however this does not work for storybook showcase, but it works in a real usecase):
```js
document.querySelector("liist-daad-filter").addEventListener("liist-daad-filter-changed", (e) => {
  console.log(e.detail.categories);
  console.log(e.detail.serviceTypes);
});
```
