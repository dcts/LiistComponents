# `<liist-author>`

A component that displays an avatar image and the name of the author.

### Default User
<Preview>
  <liist-author></liist-author>
</Preview>

### Add Name and Avatar
<Preview>
  <liist-author
    displayName="Thomas S."
    userImageUrl="https://avatars0.githubusercontent.com/u/44790691?s=460&u=2a44ea33f69d6cf02bc659381a70ea4fedf62ffb&v=4"
  ></liist-author>
</Preview>

## Properties

- **`displayName`**: Full name of the user.
- **`userImageUrl`**: URL to avatar image of user.
