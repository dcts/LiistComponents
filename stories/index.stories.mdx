# `<liist-bttn>`

Main button component. Fontawesome icon identifier and the content of the text can be parsed by adding properties `text` `faSpec` `bgColor` and `color`.

### Preview
<Preview>
  <Story name="liist-bttn">
    {html`
      <div style="display: flex; justify-content: space-around;">
        <!-- EXAMPLE 1 -->
        <liist-bttn
          color="white"
          bgColor="black"
          text="Send Suggestion"
          faSpec="far fa-envelope"
        ></liist-bttn>
        <!-- EXAMPLE 2 -->
        <liist-bttn
          color="white"
          bgColor="#9697fb"
          text="Open in Google Maps"
          faSpec="fas fa-location-arrow"
        ></liist-bttn>
        <!-- EXAMPLE 3 -->
        <liist-bttn
          color="white"
          bgColor="#76d672"
          text="Call Restaurant"
          faSpec="fas fa-phone"
        ></liist-bttn>
      </div>
    `}
  </Story>
</Preview>
