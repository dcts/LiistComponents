# `<liist-fake-map-bttn>`

A component that displays a map with a center location, an open map button and some randomly assigned liist-tags.

## Properties

| Property           | Description                                                    | Default     |
| ------------------ | -------------------------------------------------------------- | ----------- |
| `lat`              | latitude of the center of the map.  | `-73.935242` (NY) |
| `lng`              | longitude of the center of the map.  | `40.73061` (NY) |
| `width`            | width of the whle component. Can be in `px`, `%` or viewport.  | `400px` |
| `height`           | height of the whle component. Can be in `px`, `%` or viewport.  | `200px` |
| `theme`            | liist-color theme. See LiistColors documentation for all possible values. Overrides `color` and `bgColor` | `viiolet80` |
| `color`            | Tint color. Only applied when theme is not set. | `undefined` |
| `bgColor`          | Background color. Only applied if theme is not set. | `undefined` |
| `liistTitle`       | title of the list. Determines the random points.  | `""` |
| `url`              | targeturl when fake-map-bttn is clicked. | `undefined` |
| `googleMapsApiKey` | provide API key from google that has Maps Static API enabled. | `""` |
