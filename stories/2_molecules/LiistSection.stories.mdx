# `<liist-section>`

liist section component used to structure the web view

</Preview>

### Customized State
<Preview>
  <liist-section title="50+ Spots in this Hitlist">
    <li>This is a random bullet point</li>
    <li>This is another random bullet point</li>
  </liist-section>
</Preview>


## Properties
There are no properties for this component but 2 Slots of which the content can be changed
- **`title"`**: title of the section.

## Slots
Everything inside the slot is added as content of the section (below the horizontal line).
