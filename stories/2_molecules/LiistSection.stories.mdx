# `<liist-section>`

liist section component used to structure the web view


## Properties

| property     | description            | default     |
| ------------ | ---------------------- | ----------- |
| **`title`**  | title of the section.  | `loading...`|
| **`width`**  | width of the component, can be specified as pixels (e.g. `250px`) or percentage (e.g. `50%`). | `100%`|

<br></br>
<br></br>

### Default State
<Preview>
  <liist-section></liist-section>
</Preview>

### With Content
All content that is put inside the component, is rendered in the content section (**slot**) of the component.
<Preview>
  <liist-section title="50+ Spots in this Hitlist">
    <liist-place-card></liist-place-card>
    <liist-place-card></liist-place-card>
  </liist-section>
</Preview>

### With Custom Size (in px or %)
Custom size caps the overall size of the component. Is either specified as **pixel** or **percentage**. For example `width="250px"` or `width="100%"`. Default is 100%.
<Preview>
  <liist-section width="250px" title="50+ Spots in this Hitlist">
    <liist-place-card></liist-place-card>
    <liist-place-card></liist-place-card>
  </liist-section>
</Preview>
