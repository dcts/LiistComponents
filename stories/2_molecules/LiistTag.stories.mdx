# `<liist-tag>`

Liist tags are a combination of a selected icon and a LiistColor theme.

## Properties

| Property | Description            | Default     |
| -------- | ---------------------- | ----------- |
| `icon`   | name of the icon. See LiistTag (section tag-icons) for all availible tags | `pin` |
| `size`   | Size of the tag, specified in px or percentage. | `42px` |
| `theme`  | Color theme of the tag. Overrides `color` and `bgColor`. See LiistColors (themes). If no theme, no color, and no bgColor is set, the theme `viiolet80` will be assigned. | `viiolet80` |
| `bgColor`| Background color of the tag. Only applied if there is no theme. | `undefined` |
| `color`  | Tint color of the tag. Only applied if there is no theme. | `undefined` |
