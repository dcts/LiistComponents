# `<liist-tag>`

Liist tags are a combination of a selected icon and a LiistColor theme.

## Properties

| Property | Description            | Default     |
| -------- | ---------------------- | ----------- |
| `icon`   | name of the icon. See LiistTag (section tag-icons) for all availible tags | `loading...` |
| `theme`  | Color theme of the tag. See LiistColors (themes) | `viiolet80` |
| `size`   | Size of the tag, specified in px or percentage. | `42px` |
