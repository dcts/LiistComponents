# `<liist-author>`

A component that displays an avatar image and the name of the author.

## Properties

| property           | description                                                    | default     |
| ------------------ | -------------------------------------------------------------- | ----------- |
| `displayName`      | Name of the User. Inputtet text will be displayed after `By`.  | `LIIST Bot` |
| `userImageUrl`     | URL to avatar image of user.                                   | see below (# Default User) |
| `color`            | Color of the font. Supports LiistColors.                       | `smokii`    |
