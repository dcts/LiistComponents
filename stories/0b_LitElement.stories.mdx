# Lit Element Basics

Lit-Element is a lightweiht wrapper that allows to build vanilla js webcomponents with ease. It comes with some very basic but usefull functionality. It's the evolution of the polymer project maintained by google.

### Boilerplate Custom Element

```js
import { html, css, LitElement } from 'lit-element';

export class MyTestComponent extends LitElement {
  static get styles() {
    return css``;
  }

  static get properties() {
    return {
      propStr:    { type: String },
      propNum:    { type: Number },
      propBool:   { type: Boolean },
      propArr:    { type: Array },
      propObject: { type: Object },
    };
  }

  constructor() {
    super();
  }

  render() {
    return html``;
  }
}

window.customElements.define('my-test-component', MyTestComponent);
```

### Usage

```html
<my-test-component></my-test-component>
```
