# `<liist-check-item>`

A switch component that contains a checkmark and a text. Can have 2 states (checked or unchecked).

### Basic Usage
<Preview>
  <liist-check-item
    text="american"
    checked
  ></liist-check-item>
  <liist-check-item
    text="sushi"
    checked
  ></liist-check-item>
  <liist-check-item
    text="italian"
  ></liist-check-item>
</Preview>

<Preview>
  <liist-check-item
    text="mexican"
  ></liist-check-item>
  <liist-check-item
    text="fast food"
  ></liist-check-item>
  <liist-check-item
    text="asian"
  ></liist-check-item>
</Preview>

## Properties

- **`text`**: text that is displayed next to the check circle.
- **`checked`**: if `true` the checkcircle is filled.
