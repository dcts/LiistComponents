# `<liist-toggle-bttn>`

A switch component that contains a checkmark and a text. Can have 2 states (checked or unchecked).

## Properties

- **`text`**: text that is displayed next to the check circle.
- **`checked`**: if `true` the checkcircle is filled.
- **`width`**: number that specifys the width of the toggle button in px.

### Basic Usage
<Preview>
  <div>
    <liist-toggle-bttn
      text="delivery"
      width="400"
      checked
    ></liist-toggle-bttn>
    <br></br>
    <liist-toggle-bttn
      text="take-out / curbsite"
      width="400"
    ></liist-toggle-bttn>
    <br></br>

      width="400"
    ></liist-toggle-bttn>
  </div>
</Preview>
